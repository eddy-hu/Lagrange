Download Node.js

    https://nodejs.org/en/download/
Check Node.js' version

    node --version

install Angular Cli 

    npm install -g @angular/cli


Check Angular Cli's version

    ng --version

Create a new project

    ng new project-name


Start serve

    cd project-name
    ng serve
