Download Node.js

    https://nodejs.org/en/download/
Check Node.js' version

    node --version

install Vue-Cli 

    npm install -g vue-cli


Check Vue Cli's version

    vue --version

Create a new project

    vue init template-name project-name
    eg. vue init webpack web-app


Start serve

    cd project-name
    npm run dev
